TikManager (v1.0.0)
                        
TikManager — приложение для автоматизации загрузки видео
в несколько аккаунтов TikTok.

────────────────────────────────

Для начала работы необходимо добавить
один или несколько sessionId аккаунтов TikTok.

Пошаговая инструкция получения sessionId:
1) Войти в аккаунт TikTok в браузере на компьютере;
2) Нажать F12 и перейти во вкладку:
   Application → Storage → Cookies;
3) Найти cookie с именем `sessionid`;
4) Скопировать значение из колонки `Value`;
5) В программе TikManager открыть вкладку «Аккаунты»,
   нажать «Добавить» и вставить скопированное значение.

────────────────────────────────

Для работы приложения требуется наличие прокси-серверов.

Рекомендуемый тип:
• IPv4 Shared

Прокси можно приобрести, например, на сайте px6.me.  
При покупке доступен купон на скидку:

Купон: k0P4LWvQVu

Поддерживаются два режима работы:
1) Один прокси — один аккаунт;
2) Равномерное распределение аккаунтов
   между всеми доступными прокси.
